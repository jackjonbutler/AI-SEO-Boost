1. `buildFrontmatter(doc: MarkdownDocument): string` — Pattern 4 from RESEARCH.md:
   - Build fields object: `{ title: doc.title, url: doc.url, description: doc.description, date: new Date().toISOString().split('T')[0] }`
   - Filter out empty string values (don't emit `description: ""`)
   - Escape double quotes inside values: `v.replace(/"/g, '\\"')`
   - Emit lines: `${key}: "${escapedValue}"` — always double-quoted for safety (handles colons, hash, leading spaces)
   - Wrap in `---\n{lines}\n---\n\n`
   - Example output for a page with title "Home" and url "https://ex.com/":
     ```
     ---
     title: "Home"
     url: "https://ex.com/"
     date: "2026-04-20"
     ---
     
     ```

2. `urlToSlug(pageUrl: string): string` — Pattern 3 from RESEARCH.md + Pitfall 8 (home-page edge case):
   - Try `new URL(pageUrl)`:
     - Take `u.pathname`
     - Strip `.html` or `.htm` suffix: `.replace(/\.html?$/i, '')`
     - Strip leading slash: `.replace(/^\//, '')`
     - Strip trailing slash: `.replace(/\/$/, '')`
     - If result is empty OR equals 'index' → return `'index'` (home case — flat write)
     - Strip `/index` suffix if present (so `/services/index.html` → `services`, not `services/index`)
     - Return result
   - On URL parse failure → return `'index'` (fallback)

3. `buildMarkdownMirror(doc: MarkdownDocument): { slug: string; content: string }`:
   - `slug = urlToSlug(doc.url)`
   - `content = buildFrontmatter(doc) + doc.markdown.trimEnd() + '\n'` (POSIX newline, matches 03-02 decision)
   - Return `{ slug, content }`

Constraints:
- No I/O inside buildMarkdownMirror — the handler owns writeFile/mkdir.
- Don't import stripChrome or convertToMarkdown — docs arrive already processed.
- If doc.markdown is empty string, still emit frontmatter + blank body (a valid empty page).
- The `date` field is today's date (generation time), not file mtime — matches sitemap Pattern 7 decision.

Handler body:
```ts
async ({ target, outputDir }) => {
  try {
    if (!target || typeof target !== 'string' || target.trim().length === 0) {
      return { content: [{ type: 'text' as const, text: 'Error: target must be a non-empty string (URL or absolute local folder path)' }], isError: true };
    }
    if (!outputDir || typeof outputDir !== 'string' || outputDir.trim().length === 0) {
      return { content: [{ type: 'text' as const, text: 'Error: outputDir must be a non-empty string (absolute directory where mirror files will be written)' }], isError: true };
    }

    const t = target.trim();
    const dir = outputDir.trim();
    const isUrl = t.startsWith('http://') || t.startsWith('https://');
    const results = isUrl ? await crawlUrl(t) : await acquireLocal(t);
    const docs: MarkdownDocument[] = results.filter((r): r is MarkdownDocument => !isAcquisitionError(r));

    if (docs.length === 0) {
      return { content: [{ type: 'text' as const, text: `Error: no pages acquired from ${t} — nothing to mirror` }], isError: true };
    }

    // Slug collision handling (RESEARCH.md Pitfall 3)
    const writtenSlugs = new Set<string>();
    const disambiguate = (slug: string): string => {
      if (!writtenSlugs.has(slug)) { writtenSlugs.add(slug); return slug; }
      let n = 2;
      while (writtenSlugs.has(`${slug}-${n}`)) n++;
      const unique = `${slug}-${n}`;
      writtenSlugs.add(unique);
      return unique;
    };

    const limit = pLimit(5); // RESEARCH.md Open Question 2 resolution
    const writes = docs.map((doc) => limit(async () => {
      const { slug, content } = buildMarkdownMirror(doc);
      const finalSlug = disambiguate(slug);
      // Home page (slug === 'index') writes to outputDir/index.md — flat, not nested (Pitfall 8)
      const filePath = finalSlug === 'index'
        ? path.join(dir, 'index.md')
        : path.join(dir, finalSlug, 'index.md');
      await mkdir(path.dirname(filePath), { recursive: true });
      await writeFile(filePath, content, 'utf-8');
      return filePath;
    }));
    const written = await Promise.all(writes);
    return { content: [{ type: 'text' as const, text: `${written.length} markdown mirror(s) written under ${dir}` }] };
  } catch (err) {
    const msg = err instanceof Error ? err.message : String(err);
    return { content: [{ type: 'text' as const, text: `Error: ${msg}` }], isError: true };
  }
},
```

Note on the disambiguation Set: it must live INSIDE the handler (per-call scope) — never module-level — so each tool call starts fresh.
