Create src/generators/files/sitemap-xml.ts exporting `export function buildSitemapXml(docs: MarkdownDocument[], baseUrl: string): string`.

Import type: `import type { MarkdownDocument } from '../../types/index.js'` (Node16 ESM .js suffix required).

Internal helpers (not exported — keep file self-contained):

1. `escapeXml(str: string): string` — Replace in strict order: `&` → `&amp;` (MUST come first to avoid double-escaping), `<` → `&lt;`, `>` → `&gt;`, `"` → `&quot;`. Pattern from RESEARCH.md Pattern 5.

2. `resolveToAbsolute(docUrl: string, baseUrl: string): string` — Normalise file:// URLs to https:// URLs by extracting the pathname and rebasing on baseUrl. Algorithm:
   - Strip trailing slash from baseUrl: `const base = baseUrl.replace(/\/$/, '')`
   - Parse docUrl with `new URL(docUrl)`
   - If protocol is `file:`, treat pathname as relative path: extract the basename or relative slug. SIMPLEST approach: use the path AFTER the source folder root. Since the generator does not receive the source folder, use the LAST path segment and .html stripped: `path.basename(u.pathname).replace(/\.html?$/, '')`. Home case: if pathname ends in `/index.html` or equals `/`, emit `${base}/`. Else emit `${base}/${basename-without-ext}`.
   - If protocol is `http:` or `https:`, return the original URL (already absolute).
   - Import `path` via `import path from 'node:path'` for basename extraction.

3. `scorePriority(pageUrl: string): number` — Per RESEARCH.md Pattern 6 and phase spec:
   - Parse `new URL(pageUrl).pathname`
   - `segments = pathname.split('/').filter(Boolean)`
   - `if (segments.length === 0) return 1.0` (home)
   - `slug = segments.join('/').toLowerCase()`
   - `serviceKeywords = ['service', 'wrap', 'tint', 'install', 'repair', 'product', 'offering']`
     if any service keyword appears in slug → return 0.9
   - `infoKeywords = ['about', 'faq', 'pricing', 'price', 'contact', 'location', 'gallery']`
     if segments.length === 1 AND any info keyword in slug → return 0.8
   - Fall through → return 0.7

Main function body:
- `const base = baseUrl.replace(/\/$/, '')`
- `const today = new Date().toISOString().split('T')[0]` — YYYY-MM-DD (RESEARCH.md Pitfall 7)
- For each doc, compute `loc = escapeXml(resolveToAbsolute(doc.url, base))`, `priority = scorePriority(loc).toFixed(1)`
- Emit XML structure EXACTLY (RESEARCH.md Code Examples section):
  ```
  <?xml version="1.0" encoding="UTF-8"?>
  <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
    <url>
      <loc>{loc}</loc>
      <lastmod>{today}</lastmod>
      <priority>{priority}</priority>
    </url>
    ...
  </urlset>
  ```
- Finish with POSIX newline: `.trimEnd() + '\n'` (RESEARCH.md Anti-patterns — matches 03-02 decision)

Constraints:
- No Zod imports (types only — matches 03-02 pattern)
- No fs imports (pure function — RESEARCH.md Pattern 1)
- xmlns MUST be `http://www.sitemaps.org/schemas/sitemap/0.9` (the namespace URI — http is correct per spec even though we emit https URLs)
- Do NOT import cheerio, turndown, p-limit — none are needed
- If `docs` is empty, still emit a valid `<urlset>` with no `<url>` children — don't throw

Handler body (follows RESEARCH.md Pattern 2 — dual acquisition):
```ts
async ({ target, baseUrl, outputPath }) => {
  try {
    // Input validation — mirror the audit_ai_seo / generate_llms_txt pattern
    if (!target || typeof target !== 'string' || target.trim().length === 0) {
      return { content: [{ type: 'text' as const, text: 'Error: target must be a non-empty string (URL or absolute local folder path)' }], isError: true };
    }
    if (!baseUrl || typeof baseUrl !== 'string' || baseUrl.trim().length === 0) {
      return { content: [{ type: 'text' as const, text: 'Error: baseUrl must be a non-empty string (e.g. https://example.com)' }], isError: true };
    }
    if (!outputPath || typeof outputPath !== 'string' || outputPath.trim().length === 0) {
      return { content: [{ type: 'text' as const, text: 'Error: outputPath must be a non-empty string (absolute path where sitemap.xml should be written)' }], isError: true };
    }

    const t = target.trim();
    const isUrl = t.startsWith('http://') || t.startsWith('https://');
    const results = isUrl ? await crawlUrl(t) : await acquireLocal(t);
    const docs: MarkdownDocument[] = results.filter((r): r is MarkdownDocument => !isAcquisitionError(r));

    if (docs.length === 0) {
      return { content: [{ type: 'text' as const, text: `Error: no pages acquired from ${t} — nothing to include in sitemap` }], isError: true };
    }

    const xml = buildSitemapXml(docs, baseUrl.trim());
    await writeFile(outputPath.trim(), xml, 'utf-8');
    return { content: [{ type: 'text' as const, text: `sitemap.xml written to ${outputPath.trim()} (${docs.length} URLs, ${xml.length} bytes)` }] };
  } catch (err) {
    const msg = err instanceof Error ? err.message : String(err);
    return { content: [{ type: 'text' as const, text: `Error: ${msg}` }], isError: true };
  }
},
```
