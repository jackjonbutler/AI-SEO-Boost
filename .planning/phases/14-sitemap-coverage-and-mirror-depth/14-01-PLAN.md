**`fetchUrlsFromSitemap(sitemapUrl: string): Promise<string[]>`**
- Fetches the given URL with `AbortSignal.timeout(5000)`
- Returns `[]` on non-200 status or fetch error
- Parses with `cheerio.load(xml, { xml: true })`
- Returns `$('url loc').map((_, el) => $(el).text().trim()).toArray()`

**`fetchSitemapUrls(origin: string): Promise<string[] | null>`**
- Fetches `${origin}/sitemap.xml` with `AbortSignal.timeout(5000)`
- Returns `null` on fetch error (catches the exception — `null` means no sitemap / unreachable)
- Returns `null` if response status is not 200 (e.g. 404)
- Parses with `cheerio.load(xml, { xml: true })`
- COV-02: If `$('sitemapindex').length > 0`, extracts child sitemap URLs via `$('sitemap loc').map((_, el) => $(el).text().trim()).toArray()`. If empty array, returns `[]`. Otherwise calls `fetchUrlsFromSitemap(childSitemapUrls[0])` and returns the result.
- Otherwise (regular urlset): returns `$('url loc').map((_, el) => $(el).text().trim()).toArray()`
- IMPORTANT: Use `{ xml: true }` — do NOT use `{ xmlMode: true }` (deprecated in cheerio 1.2.0)
- IMPORTANT: Use lowercase selectors `$('sitemapindex')`, `$('sitemap loc')`, `$('url loc')` — XML mode is case-sensitive

**`sampleUrls(urls: string[], maxSample: number): string[]`**
- If `urls.length <= maxSample` return `urls` as-is
- Otherwise, evenly spread: `Array.from({ length: maxSample }, (_, i) => urls[Math.floor(i * (urls.length / maxSample))])`

**`toMdUrl(pageUrl: string): string`**
- Parses `pageUrl` as `new URL(pageUrl)`
- Home page (`pathname === '/'`): returns `${u.origin}/index.md`
- All other pages: strips trailing slash from pathname with `.replace(/\/$/, '')`, then appends `/index.md`
- Returns `${u.origin}${pathname}`
- PITFALL: Do NOT skip the trailing slash strip — `/about/` would produce `/about//index.md`

**`hasMdMirror(pageUrl: string): Promise<boolean>`**
- Calls `toMdUrl(pageUrl)` to get the mirror URL
- HEAD-probes with `fetch(mdUrl, { method: 'HEAD', signal: AbortSignal.timeout(4000) })`
- Returns `true` if status === 200, `false` otherwise
- On fetch error (catch), returns `false`

**New URL branch logic:**

```
const MAX_SAMPLE = 20;
const origin = originFor(target);
const allSitemapUrls = await fetchSitemapUrls(origin);

if (allSitemapUrls === null) {
  // null = no sitemap / unreachable — graceful fallback (COV-01, success criterion 4)
  return {
    dimension,
    status: 'warning',
    severity: 'medium',
    message: `No sitemap found at ${origin}/sitemap.xml — mirror coverage cannot be estimated.${buildMarkdownPlacementNote(framework)}`,
    suggestedToolCall: 'generate_markdown_mirrors',
  };
}

const sampled = sampleUrls(allSitemapUrls, MAX_SAMPLE);
const mirrorResults = await Promise.all(sampled.map(url => hasMdMirror(url)));
const mirrored = mirrorResults.filter(Boolean).length;
const total = sampled.length;

if (total === 0) {
  // Sitemap found but has zero parseable URLs
  return {
    dimension,
    status: 'warning',
    severity: 'medium',
    message: `Sitemap found at ${origin}/sitemap.xml but contains no parseable URLs — mirror coverage cannot be estimated.${buildMarkdownPlacementNote(framework)}`,
    suggestedToolCall: 'generate_markdown_mirrors',
  };
}

const pct = Math.round((mirrored / total) * 100);
const coverageLabel = `${mirrored}/${total} sampled URLs have a mirror — estimated ${pct}% coverage`;

if (pct === 100) {
  return { dimension, status: 'pass', severity: 'low', message: coverageLabel };
}
if (pct === 0) {
  return {
    dimension,
    status: 'fail',
    severity: 'medium',
    message: `${coverageLabel}.${buildMarkdownPlacementNote(framework)}`,
    suggestedToolCall: 'generate_markdown_mirrors',
  };
}
// 1–99%: partial mirrors → warning
return {
  dimension,
  status: 'warning',
  severity: 'medium',
  message: `${coverageLabel}.${buildMarkdownPlacementNote(framework)}`,
  suggestedToolCall: 'generate_markdown_mirrors',
};
```
